# Build Phases

## Phase 1 — Project setup ✅
GitHub repository created. Folder structure established.
Documentation framework in place.

## Phase 2 — Infrastructure as Code
CloudFormation templates for all AWS resources.
Stored in /infrastructure folder.

## Phase 3 — CI/CD Pipeline
AWS CodePipeline connected to GitHub.
CodeBuild configured for frontend and API builds.

## Phase 4 — Authentication
AWS Cognito user pool and identity pool.
Google and Facebook social sign-in.
Custom buyer/seller role attribute.

## Phase 5 — Frontend scaffold
React app created with Vite.
Routing set up for buyer and seller views.
Cognito authentication integrated.

## Phase 6 — API and Database
DynamoDB tables created via CloudFormation.
Lambda functions for core API routes.
API Gateway configuration.

## Phase 7 — Buyer dashboard
Home feed with AWS Personalize recommendations.
Product pages with size, colour, reviews.
Cart functionality.

## Phase 8 — Seller dashboard
Inventory management.
Performance dashboard from RDS analytics.
Product upload with S3 image storage.

## Phase 9 — CDN and media
CloudFront distribution for S3 assets.
Image upload flow from seller dashboard.

## Phase 10 — Testing and deployment
End to end testing.
Final production deployment.
Medium article series completed.
